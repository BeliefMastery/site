# Glossary — shared terms for assessments and reports

Use this file as the **single canonical gloss** for framework and clinical terms that appear in more than one engine or tool shell. When you fix a readability issue (e.g. D-*, C-*), add or update entries here and link from report copy where helpful.

**Process:** First use in a given report may use a **one-line gloss** inline; repeated tools should align wording with this table. See [STYLE_ADDENDUM.md](./STYLE_ADDENDUM.md) and [STRATEGY_V2.md](./STRATEGY_V2.md) (Pass 2 jargon gate).

## How to add an entry
