Belief Mastery & Sovereign of Mind — static diagnostics platform (https://beliefmastery.github.io/site/). Production GitHub Pages site for Warwick Marshall's books plus a coordinated toolkit of structured self-assessments focused on mental sovereignty, cognitive defense, worldview clarity, and authorship under pressure.

What it does: zero-build HTML/CSS/vanilla ES modules—no framework runtime. Users browse catalog and author context, then run assessments that walk phased questionnaires, score against curated data modules, render narrative and structured results, and export JSON/CSV (and executive-style briefs where implemented) with versioning metadata. The shell delivers consistent navigation, responsive layout, accessibility patterns, theme and font scaling, and optional WebGL background treatment via shared scripts.

Engines: eleven client-side assessment engines, each owning UX flow and scoring for one tool—diagnosis-engine (Pathology Assessment), coaching-engine (Life Domain Review), needs-dependency-engine (Dependency Loop Tracer), sovereignty-spectrum-engine (Your Sovereignty Paradigm), paradigm-engine (Logos Structure), manipulation-engine (Manipulation Defense Decoder), sovereignty-engine (Cognitive Resistance Capacity), channels-engine (Channel Flow Diagnostics), character-sheet-engine (Astrological Character Sheet), entities-engine (Will Anomaly Mapping), outlier-aptitude-engine (Aptitude Mapping). Cross-cutting utilities live in shared/. A twelfth offering, Unlocked GPT, links to an external ChatGPT custom GPT—no in-repo engine by design.

Stack & stance: semantic HTML, CSS custom properties, ES modules, local/session storage for progress where applicable—readable, auditable client architecture suitable for portfolio review, research exports, or future native packaging.
