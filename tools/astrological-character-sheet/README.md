## What This Tool Is

**Astrological Character Sheet: D&D-Style Character Generator**

A tool that generates a D&D-style character sheet based on your astrological profile from Western, Chinese, and Mayan astrology systems.

## Description

Transform your astrological profile into a D&D-style character sheet combining Western, Chinese, and Mayan astrology for fun roleplay games. This playful tool translates astrological data into character attributes for gaming purposes.

## Key Features

- Combines Western, Chinese, and Mayan astrology
- Generates D&D-style character attributes
- Fun roleplay game integration
- Export character sheet as JSON (optional)

## How to Use

1. Visit https://beliefmastery.github.io/site/character-sheet.html
2. Enter your Western astrology details (birth date/time)
3. Enter your Chinese astrology details
4. Enter your Mayan astrology details (optional)
5. Review your generated character sheet
6. Export character sheet as JSON (optional)
