# Pathology Assessment

## Live Tool
**https://beliefmastery.github.io/site/diagnosis.html**

## What This Tool Is

**Pathology Assessment: DSM-5 Self-Reporting Diagnostic Tool**

An educational tool for exploring mental health diagnostic categories through structured self-reporting questionnaires based on DSM-5 criteria.

## Description

Pathology assessment and clinical education mapping system based on DSM-5 criteria. For educational and self-reflection purposes only.

## Key Features

- 100+ disorders across 13+ major diagnostic categories
- Category selection guide to identify relevant areas
- Comorbidity detection for co-occurring conditions
- Comprehensive treatment database
- Multi-branching logic for complex cases
- Data export (JSON) for professional consultation

## Important Note

**This system is for educational and informational purposes only. It is not a substitute for professional mental health evaluation.**

## How to Use

1. Visit https://beliefmastery.github.io/site/diagnosis.html
2. Complete the category selection guide
3. Answer detailed symptom questionnaires
4. Review your assessment results
5. Export results as JSON (optional)

## Technical Details

- Client-side JavaScript (no server required)
- All data stored locally
- Results exportable as JSON
- Based on DSM-5 diagnostic criteria
