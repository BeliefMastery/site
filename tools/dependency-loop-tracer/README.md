## What This Tool Is

**Dependency Loop Tracer: Needs and Vices Assessment**

A tool that identifies dependency loops through needs and vices assessment. Based on Chapter 5: Needs and Beliefs from Belief Mastery.

## Description

Identify dependency loops through needs and vices assessment. Traces surface symptoms to root needs and breaks dependency cycles using STRICT PROPRIETARY needs and vices vocabulary from Belief Mastery.

## Key Features

- 4-phase architecture: Initial Screening, Loop-Specific Deep Dive, Causal Need Chain Mapping, Validation & Prioritization
- Maps 18+ dependency loops (SPACE, JOY, SECURITY, etc.)
- Links unmet needs to vices/beliefs
- Traces causal chains from symptoms to root needs
- Uses proprietary needs/vices vocabulary from Belief Mastery

## How to Use

1. Visit https://beliefmastery.github.io/site/needs-dependency.html
2. Complete the initial screening
3. Complete the loop-specific deep dive
4. Review the causal need chain mapping
5. Export results as JSON (optional)
